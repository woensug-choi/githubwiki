React app with Charkara UI components to deply github wiki as a webpage.

- Chakra ui css (https://chakra-ui.com/docs/features/css-variables#creating-scoped-theme-aware-css-variables) and github markdown cs conflits (https://github.com/KrauseFx/markdown-to-html-github-style/blob/master/convert.js)
- Uses Oracle Cloud personal VPN to overcome CORS limitations when fetching github wiki
